# discordJsBot-

Template to create your own bots 

A discord bot using JS with NodeJs

feel free to update or suggest new things.

The previous knowlege you need is JS

## Local bot

Please create a [Discord](https://discord.com/) account first

Then go to the [Discord Developer](https://discord.com/developers/applications) platform

to create your application, 

Once you've created your bot, invite it to your server with the OAuth2 url.

You need [NodeJs](https://nodejs.org) to run this bot in your machine.
